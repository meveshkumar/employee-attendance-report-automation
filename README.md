# Employee Attendance Report Automation

## Project Overview

Employee Attendance Report Automation is a Python-based automation project that reads employee attendance data from Excel, calculates attendance status, generates management reports, performs department-wise analytics, formats Excel reports professionally, and automatically emails the final report.

---

## Features

* Read attendance data from Excel
* Calculate On Time, Late, and Absent status
* Generate attendance reports automatically
* Create summary dashboards
* Generate department-wise analytics
* Format Excel reports using OpenPyXL
* Create date-based report filenames
* Send reports automatically through Gmail

---
